SQLite 3.8.11 for z/OS UNIX
==========================
This is a port of the SQLite for use on z/OS.  The files in the UNIX
directory were generated on Linux from the SQLite fossil repository.
I did the original ./configure on Linux to generate the equivalent of
the amalgamation, creating the files:
keywordhash.h , opcodes.c , opcodes.h , parse.c , parse.h , parse.y ,
shell.c , sqlite3.c , sqlite3.h , and sqlite3ext.h

I created a tar file from these files.  I then uploaded the tar file to
z/OS. I unwound the tar file into a UNIX directory using the command:

pax -ofrom=iso8859-1,to=ibm-1047 -rf "//sqlite.v3r8m11.tar"

This port was very simple due to the dilligence of the original author,
Dr. Richard Hipp, who had already implemented the EBCDIC logic. All that
I did was a bit of packaging and verification on z/OS UNIX.
