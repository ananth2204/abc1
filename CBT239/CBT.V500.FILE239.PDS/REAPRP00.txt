%OPTION  ===>_ZCMD                                                             +

%                      'PRINT'  primary command
+                      ------------------------

   Primary command%PRINT+allows you print assembler-type listing of the project
to the ISPF LIST data set.
   The format of PRINT command is:

  %P{RINT} {X{REF}}+

   If operand%XREF+specified then cross reference table will be printed.

   Once PRINT command is complete you may use ISPF LIST command to output the
data set.

   Examples:
  %PRINT      +- Print the project listing
  %P X        +- Print the project listing and the cross reference table
