Install Instructions
====================

czzCICS is distributed freely from the CBTTAPE.ORG website. To install:

1. Download file #999 from the CBT site
2. Unzip the downloaded file to extract the TSO XMIT file
3. Upload the XMIT file to your mainframe system
4. Extract the XMIT file into your.dataset.INSTALL
5. Run INST01 to extract the czzcics.LOAD library
5. Review member #USAGE for how to use czzCICS.
5. Review member EXAMP* for examples on using czzCICS.
