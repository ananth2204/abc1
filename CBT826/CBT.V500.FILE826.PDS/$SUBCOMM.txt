               SUBCOMMANDS OF THE CNCLPG PROGRAM      v 1.2.10E

 ----  -  ------------------------------------------------------------
'CSCB' -  DISPLAY ALL address spaces and ASIDs satisfying the parms
'DISP' -  DISPLAY ALL address spaces and ASIDs satisfying the parms
 ----  -  ------------------------------------------------------------
'CNCL' -  Make the address space cancelable
'NCNC' -  Make the address space non-cancelable
'SWAP' -  Make the address space swappable.
'FSWP' -  Forces the address space swappable when DONTSWAP count > 1
'NSWP' -  Make the address space non-swappable (add 1 to DONTSWAP cnt)
'TSWP' -  Do a TRANSWAP on the address space to make it non-swappable
'FORC' -  Allow the address space to be FORCED without a CANCEL first
'NFRC' -  To FORCE the address space, you must do a CANCEL first
'BURN' -  Do a CALLRTM TYPE=MEMTERM to get rid of an address space
'KILL' -  Do a CALLRTM TYPE=MEMTERM-first turn off non-memtermable bits
'NMEM' -  Turn on the main non-memtermable bit, make non-cancelable too
'MEMT' -  Turn off the main non-memtermable bit for the address space
'COUNT -  Display the DONTSWAP count of an address space
