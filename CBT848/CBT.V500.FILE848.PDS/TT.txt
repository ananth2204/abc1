   LIBSPOOL DATA SETS
  --------------------

          1) SYSX.LIBSPOOL.LIBPROD -
                          CURRENT LIBRARIAN JCL LIBRARY
                          CONTAINS THE CURRENT JCL MODULES
                          GOTTEN FROM THE LIBSPOOL TASK.

          2) SYSX.LIBSPOOL.LIBREST -
                          RESTORE FROM LIBRARIAN ARCHIVE TAPE
                          LIBRARIAN JCL LIBRARY

          3) SYSX.LIBSPOOL.ARCPDS -
                          ARCHIVE DIRECTORY
                          PARTIONED DATA SET CONTAINING INFORMATION
                          ON THE CONTENTS OF THE LIBRARIAN ARCHIVE
                          TAPES.

          4) PPODOPR.LIBSPOOL.DALYARCH -
                           DAILY LIBRARIAN ARCHIVE TAPE

          5) PPODOPR.LIBSPOOL.WKLYARCH -
                            WEEKLY LIBRARIAN ARCHIVE TAPE
