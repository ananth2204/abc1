NRequesting Resources
N--------------------
NData Set Resources
N------------------
To execute a program, you must request the data sets needed to supply
Ndata to the program and to receive output records from the program.
N
NSysout Data Set Resources
N-------------------------
NA sysout data set is a system-handled output data set. This data set
Nis placed temporarily on direct access storage. Later, at the
Nconvenience of the system, the system prints it, punches it, or sends
Nit to a specified location. Because sysout data sets are processed
Nby the system, the programmer can specify many parameters to control
Nthat processing.
