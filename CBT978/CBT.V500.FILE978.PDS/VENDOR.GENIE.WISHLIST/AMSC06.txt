NAccess Method Services for Catalogs
N-----------------------------------
NChapter 6. ALTER LIBRARYENTRY
N-----------------------------
NThe ALTER LIBRARYENTRY command modifies the attributes of an existing
Ntape library entry. Use this command to recover from tape volume
Ncatalog errors.
N
NBecause access method services cannot change the library manager
Ninventory in an automated tape library, use Interactive Storage
NManagement Facility (ISMF) for normal tape library alter functions.
