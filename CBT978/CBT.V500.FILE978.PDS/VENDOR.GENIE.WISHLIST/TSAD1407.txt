NChapter 14: Recording logrec error records
N------------------------------------------
NWhen an error occurs, the system records information about the error in
Nthe logrec data set or the logrec log stream. The information provides
Nyou with a history of all hardware failures, selected software errors,
Nand selected system conditions. Use the Environmental Record, Editing,
Nand Printing program (EREP):
N
N-> To print reports about the system records
N
N-> To determine the history of the system
N
N-> To learn about a particular error
N
