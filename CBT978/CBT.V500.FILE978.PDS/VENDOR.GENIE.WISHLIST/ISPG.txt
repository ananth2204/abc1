NInteractive System Productivity Facility (ISPF) Services Guide
N==============================================================
NPreface
N-------
This document describes how to use ISPF dialog management component (DM)
Nservices and Program Development Facility component (PDF) services.
NProgrammers who develop applications with ISPF can use the services
Ndescribed in this publication to develop dialogs from programs or
Ncommand procedures.
N
NWho should use this document?
N-----------------------------
NThis document is for application programmers who develop dialogs using
NISPF. Users should be familiar with coding in CLIST, REXX, or any of
Nthe other programming or command procedure languages supported by ISPF
Nin the MVS environment.
N
NWhat is in this document?
N-------------------------
NThis document contains two chapters.
N
NChapter 1, "Introduction to ISPF services" describes how to invoke ISPF
Nservices, provides an explanation of various service return codes, and
Nlists and summarizes all of the services described in this document.
N
NChapter 2, "Description of the ISPF services" contains this information
Nabout each of the ISPF services:
N
N-> A description of the function and operation of the service. This
N   description also refers to other services that can be used with this
N   service.
N
N-> The syntax used to code the service, showing both the command
M   procedure format and the call format.
N
N-> A description of any required or optional keywords or parameters.
N
N-> A description of the error codes returned by the service.
N
N-> Examples of the how the service is used to develop dialogs.
N
NThe services are listed in alphabetical order.
