N----------------------------------------------------------
NAppendix A. Obsolete attributes
N-------------------------------
NSome resource definition attributes do not apply to resources used in
NCICS Transaction Server for z/OS, Version 5 Release 3 , but are
Nsupported to provide CSD compatibility for earlier releases of CICS
Nwhere they are still valid.
N
NFor information about compatibility mode, see Sharing the CSD between
Ndifferent releases of CICS in Configuring.
N
NThe following attributes do not apply to resources used in this CICS
Nrelease. Check the documentation for the earlier CICS releases that
Nyou are using to see whether the attribute is supported in that
Nrelease.
