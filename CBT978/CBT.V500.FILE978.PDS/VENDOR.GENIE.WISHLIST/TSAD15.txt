NTools and Service Aids
N======================
NChapter 15: AMBLIST
N-------------------
NAMBLIST provides the following problem data:
N
N-> Formatted listing of an object module
N
N-> Map of the control sections (CSECTs) in a load module or program
N   object
N
N-> List of modifications to the code in a CSECT
N
N-> Map of all modules in the link pack areas (LPA)
N
N-> Map of the contents of the DAT-on nucleus. The map no longer
N   represents the IPL version and message AMB129I will be issued.
N
NThese formatted listings can help you diagnose problems related to
Nmodules as they currently exist on your system. AMBLIST is a batch job
Nthat runs in problem state.
