NGenuine Every Noteworthy Itemized Examplizer (GENIE)
N----------------------------------------------------
NChapter 3. Using the GENIE edit macro
N-------------------------------------
NHow to use GENIE
N----------------
NWhile editing your member you type %genie in the Command field and
Neither an A (after) or B (before) on the line where you want to
Ninsert a WISHLIST member. Then expand TOPICs and CHAPTERs to peruse
N
N1) Locate a line in your EDIT session and use a/b to set the
N   desired destination.
N
N2) Invoke the edit macro.
M   Syntax: GENIE |WISH|?|=keyword|
M           WISH    : The combination of a TOPIC, CHAPTER, and ID.
M           ?       : Pop-up the Primary Command help panel at startup.
M           =keyword: Find keyword in WISH files.
N
N   In the Command ===> %genie |WISH|?|=keyword|
N
N   If you asked for a valid WISH then your EDIT session will contain
N   the proper Data, Informational, Message, and Note lines
N   for that item. If your keyword was not located then
N   GENIE will display an empty table. You can use primary command
N   REFRESH to get the list of available TOPICS, try another keyword,
N   or SEARCH for a string.
N
I  =keyword: Is a way to invoke FIND on a keyword.
* End of WISH.
