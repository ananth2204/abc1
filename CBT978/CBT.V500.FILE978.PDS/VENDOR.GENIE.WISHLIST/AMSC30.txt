NAccess Method Services for Catalogs
N-----------------------------------
NChapter 30. REPRO
N-----------------
NThe REPRO command performs the following functions:
N
N-> Copies VSAM and non-VSAM data sets
N
N-> Copies catalogs
N
N-> Copies or merges tape volume catalogs
N
N-> Splits integrated catalog facility catalog entries between two
N   catalogs
N
N-> Splits entries from an integrated catalog facility master catalog
N   into another integrated catalog facility catalog
N
N-> Merges integrated catalog facility catalog entries into another
N   integrated catalog facility user catalog.
N
NRestriction: Access Method Services does not use RLS. If an RLS keyword
Nis specified on the DD statement of a file to be opened by AMS, the
Nkeyword will be ignored and the file will be opened and accessed in
Nnon-RLS mode.
