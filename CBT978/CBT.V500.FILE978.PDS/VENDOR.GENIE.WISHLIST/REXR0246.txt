N    Host commands and host command environments
N    -------------------------------------------
N      The ISPEXEC and ISREDIT host command environments
N      -------------------------------------------------
NThe ISPEXEC and ISREDIT host command environments are available only
Nto REXX execs that run in ISPF. Use the environments to invoke ISPF
Ncommands and services, and ISPF edit macros.
I
ITo use ISREDIT, you must be in an EDIT or VIEW session.
N
N
NISPEXEC
N-------
NISPEXEC offers you the Dialog capabilities of ISPF and its services.
N
NISREDIT
N-------
NISREDIT offers you all of the capabilities of an EDIT session.
N
MGENIE is an EDIT Macro that also uses ISPF Dialog capabilities and
MISPF services.
