NChapter 5. ATOMSERVICE resources
N--------------------------------
NAn ATOMSERVICE resource defines an Atom service, feed, collection, or
Ncategory document that CICS can deliver to a Web client over HTTP.
N
NTo support each ATOMSERVICE resource, you need a URIMAP resource to
Nhandle incoming Web client requests and point to the appropriate
NATOMSERVICE resource.
