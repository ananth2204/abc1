NChapter 2. REXX general concepts
N--------------------------------
N  Assignments and symbols
N  -----------------------
N    Simple symbols
N    --------------
NA simple symbol does not contain any periods and does not start with
Na digit (0-9).
N
NBy default, its value is the characters of the symbol (that is,
Ntranslated to uppercase). If the symbol has been assigned a value, it
Nnames a variable and its value is the value of that variable.
N
NExamples:
N---------
NThese are simple symbols:
N
DFred            /* The initial value of Fred is FRED */
DWhatagoodidea? /* Same as WHATAGOODIDEA? */
D?12
