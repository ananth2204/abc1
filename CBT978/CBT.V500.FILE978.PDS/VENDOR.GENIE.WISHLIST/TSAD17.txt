N======================
NChapter 17: AMATERSE
N--------------------
NAMATERSE is a service aid program that operates in problem state. You
Ncan use AMATERSE to pack a data set before transmitting a copy to
Nanother site, typically employing FTP as the transmission mechanism. A
Ncomplementary unpack service is provided to create a similar data set
Nat the receiving site.
