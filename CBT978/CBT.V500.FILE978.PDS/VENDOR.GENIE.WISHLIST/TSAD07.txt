NChapter 7: The dump grab bag
N----------------------------
NA dump contains information about an error that can help you identify
Na problem type. Using interactive problem control system (IPCS), the
Ninformation about the error is formatted to provide a quick and
Neffective method of retrieval.
N
NThe hints that follow apply to processing all kinds of dumps: SVC
Ndumps, stand-alone dumps, and SYSMDUMP dumps.
N
N
NMajor topics cover the following:
N
N Problem data for storage overlays
N Problem data from the linkage stack
N Problem data for modules
N Problem data from recovery work areas
N Problem data for ACR
N Problem data for machine checks
