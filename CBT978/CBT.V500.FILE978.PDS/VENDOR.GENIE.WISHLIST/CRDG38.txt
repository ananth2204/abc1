N----------------------------------------------------------
NChapter 38. The resource definition batch utility DFHCSDUP
N----------------------------------------------------------
NThe CICS system definition utility program, DFHCSDUP, is a component
Nof resource definition online (RDO). DFHCSDUP is an offline utility
Nprogram that allows you to read from and write to a CICS system
Ndefinition (CSD) file, either while CICS is running or while it is
Ninactive.
