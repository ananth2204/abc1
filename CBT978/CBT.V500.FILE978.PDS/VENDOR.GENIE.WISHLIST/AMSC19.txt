NChapter 19. DEFINE USERCATALOG|MASTERCATALOG
N--------------------------------------------
NThe DEFINE USERCATALOG command defines a user catalog. When you use
Nthis command, you can specify attributes for the catalog as a whole
Nand for the components of the catalog.
N
NUSERCATALOG is comprised of three components:
N  DEFINE -
N    USERCATALOG|MASTERCATALOG( -
N      parameters -
N    ) -
N    DATA( -
N      parameters -
N    ) -
N    INDEX( -
N      parameters -
N    ) /* - */
