N======================
NChapter 20: Messages
N--------------------
NReceiving messages
N------------------
NThe system issues messages through WTO and WTOR macros to the following
Nlocations. Routing codes determine the display and print location of
Nthe messages.
N
N-> Console
N
N-> Extended console
N
N-> Hard-copy log
N
N-> Job log
N
N-> SYSOUT data set
N
NThe system issues messages to the SYSLOG and OPERLOG using the WTL
Nmacro.
