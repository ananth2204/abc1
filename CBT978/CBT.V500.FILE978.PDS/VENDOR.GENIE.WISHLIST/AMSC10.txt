NAccess Method Services for Catalogs
N-----------------------------------
NChapter 10. CREATE VOLUMEENTRY
N------------------------------
NThe CREATE VOLUMEENTRY command creates tape volume entries. Use this
Ncommand only to recover from tape volume catalog errors. Because access
Nmethod services cannot change the library manager inventory in an
Nautomated tape library, ISMF should be used for normal tape library
Ncreate functions.
