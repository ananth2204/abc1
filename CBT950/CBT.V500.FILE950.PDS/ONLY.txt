/* Name:       ONLY                                                   */
/* Short:      ONLY (See INCL)                                        */
/* Typ:        REXX-Macro                                             */
/* Author:     Norbert Haas, noaDatex.de                              */
/* Function:   Excludes all line, that do not contain <parm> and      */
/*             hides all excludet lines                               */
/* Syntax:     ONLY parm Ýoptional parms¨                             */
/* Parameter:  parm = string to find                                  */
/*             optional parm = parms used with findcommand            */
/* Example:    ONLY dd word                                           */
/*                                                                    */
/* Version   Autor         Datum     Anlass                           */
/* --------- ------------- --------- -------------------------------- */
/* V01.01    N.Haas        30.05.03  english version                  */
/* V01.00    N.Haas        22.12.92  Erstellung                       */
/**********************************************************************/
"ISREDIT MACRO (parm)"

"ISREDIT EXCLUDE ALL"
"ISREDIT FIND ALL" parm
"ISREDIT HIDE X"

EXIT
