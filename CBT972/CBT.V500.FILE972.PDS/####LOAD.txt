
GENIE.PDS(####LOAD)

The following tasks will receive individual TOPIC(S) into one WISHLIST
data set. The following TOPIC(S) have been created. Some contain a sparse
ITEM(S) list, whilst others are quite populated. Sample and Example usage
within ITEM(S) come either from the manul or were provided by the TOPIC
editor.

  DASM - Data And Storage Management.
    The home for IEFBR14.

  DFPU - Data Facility Product User Guide.
    IBM Utilities:
      IEBCOMPR -Compare Data Sets Program.
      IEBCOPY -Library Copy Program.
      IEBDG -Test Data Generator Program.
      IEBEDIT -Edit Job Stream Program.
      IEBGENER -Sequential Copy/Generate Data Set Program.

      IEBISAM Indexed Sequential Access Method Program. Unsupported!

      IEBPTPCH -Print-Punch Program.
      IEBUPDTE -Update Data Set Program.
      IEHINITT -Initialize Tape Program.
      IEHLIST -List System Data Program.
      IEHMOVE -Move System Data Program.
      IEHPROGM -Program Maintenance Program.

  IBMK - IBM Manuals to be used for WISHes.
    A long list of manuals that could become TOPIC(S) for GENIE.

  JCLR - JCL Reference Manual.
    Each statement and attribute broken out with valid options.

  MITR - z/OS MVS Initialiation and Tuning.
    Just the IEASYMxx member at this time.

  PMAU - SMP/E Planning and Migration Assistant.
    Setting up to use PMA.
    Running PMA reports.

  PSFC - Print Services Facility: Customization.
    Additional JCL constructs specific to Print Services Facility.

  REXR - REXX Reference.
    Instructions and Built-in Functions.

  RXCP - REXX Compiler Presentation -George Fulk.
    How to run the REXX Compiler and some sample REXX code.

  RXCU - REXX Compiler -User's Guide and Reference.
    Compiler options and control directives.

  TSAD - Tools And Service Aids.
    System Programmer Utilities and programming techniques.
