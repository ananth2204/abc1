NChapter 10. TSO/E REXX commands
N-------------------------------
NTS
N--
NTS (Trace Start) is an immediate command you can use to start tracing
NREXX execs. Tracing lets you control the execution of an exec and debug
Nproblems. The TS immediate command is available if an exec is running
Nin TSO/E and you press the attention interrupt key to enter attention
Nmode. You can enter TS in response to the REXX attention prompting
Nmessage, IRX0920I. The exec continues processing and tracing is
Nstarted.
N
NSyntax
D  TS
