NChapter 10. TSO/E REXX commands
N-------------------------------
NHT
N--
NHT (Halt Typing) is an immediate command you can use to suppress
Nterminal output that an exec generates. The HT immediate command is
Navailable only if an exec is running in TSO/E and you press the
Nattention interrupt key to enter attention mode. You can enter HT in
Nresponse to the REXX attention prompting message, IRX0920I.
N
NSyntax
D  HT
