NInteractive System Productivity Facility (ISPF) Services Guide
N==============================================================
NChapter 1. Introduction to ISPF services
N----------------------------------------
NA summary of the ISPF services
N------------------------------
NVariable services
N-----------------
NAll Functions
N-------------
NVERASE:   Removes variables from the shared pool or profile pool.
N
NVGET:     Retrieves variables from the shared pool or profile pool or
N          retrieves the value of a system symbolic variable.
N
NVPUT:     Updates variables in the shared pool or profile pool.
N
NProgram Functions Only
N----------------------
NVCOPY:    Copies data from a dialog variable to the program.
N
NVDEFINE:  Defines function program variables to ISPF.
N
NVDELETE:  Removes the definition of function variables.
N
NVMASK:    Associates a mask with a dialog variable.
N
NVREPLACE: Updates dialog variables with program data specified in the
N          service request.
N
NVRESET:   Resets function variables.
