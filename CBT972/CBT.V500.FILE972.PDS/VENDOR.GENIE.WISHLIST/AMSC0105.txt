NUnderstanding the Order of Assigned Data Set Attributes
N-------------------------------------------------------
NYou can select attributes in more than one way with the DEFINE
Ncommand. Because more than one value for the same attribute can be
Ngiven, attributes are selected in the following order of precedence:
N
N1. Explicitly specified attributes
N
N2. Modeled attributes
N
N3. DATA CLASS attributes
N
N4. Access method services command defaults
* End of wish.
