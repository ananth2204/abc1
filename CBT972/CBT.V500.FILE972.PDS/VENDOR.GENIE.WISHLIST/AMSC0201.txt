NAccess Method Services for Catalogs

NChapter 2. Modal Commands
N-------------------------
NModal Commands
N--------------
NYou cannot use these commands when Access Method Services is running
nin Time Sharing Option (TSO).
N
N-> IF-THEN-ELSE command sequence, which controls command execution on
N   the basis of condition codes
N
N-> NULL command, which causes the program to take no action
N
N-> DO-END command sequence, which specifies more than one functional
N   access method services command and its parameters
N
N-> SET command, which resets condition codes
N
N-> CANCEL command, which ends processing of the current job step
N
N-> PARM command, which chooses diagnostic aids and options for
N   printed output.
* End of wish.
