NAccess Method Services for Catalogs
N-----------------------------------
NChapter 31. Using SETCACHE
N--------------------------
NCase 1: Setting Caching On for the Subsystem
N--------------------------------------------
NSETCACHE SUBSYSTEM ON enables normal caching for the subsystem. If
Ncache storage is disabled or in a pending state, the command fails.
N
NNote: Pinned data, a cache failure, or cache disabled for maintenance
Ncan cause the command to fail.
N
NAbbreviation: SETC
