NChapter 38. The resource definition batch utility DFHCSDUP
N==========================================================
NInvoking DFHCSDUP as a batch program
N------------------------------------
NThe DFHCSDUP PROCESS command
N----------------------------
NApply maintenance to the CSD file for a specific APAR.
N
NSyntax
N------
D  PROCESS Apar ( aparnumber )
N
NOptions
N-------
N    Apar(aparnumber): The number of the APAR providing the maintenance;
N                      for example, PROCESS APAR(PQ12417) is used to
N                      apply maintenance for APAR PQ12417.
* End of wish.
