NChapter 16. JVMSERVER resources
N===============================
NA JVMSERVER resource defines the runtime environment for a JVM server.
N
NThe JVMSERVER resource defines the location of the JVM profile and the
NLanguage Environment options that are required to create a Language
NEnvironment enclave and a JVM server in a CICS region.
