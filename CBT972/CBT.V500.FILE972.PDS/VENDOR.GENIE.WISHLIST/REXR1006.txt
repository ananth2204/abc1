NChapter 10. TSO/E REXX commands
N-------------------------------
NHI
N--
NHI (Halt Interpretation) is an immediate command you can use to halt
Nthe interpretation of all currently executing execs. The HI immediate
Ncommand is available only if an exec is running in TSO/E and you press
Nthe attention interrupt key to enter attention mode. You can enter HI
Nin response to the REXX attention prompting message, IRX0920I.
N
NSyntax
D  HI
