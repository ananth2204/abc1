NChapter 35. URIMAP resources
N============================
NURIMAP definitions are resources that match the URIs of HTTP, Atom
Nfeed, or web service requests, and provide information on how to
Nprocess the requests.
N
NThe following sub-topics can be found in the manual:
N
N-> Requests from a web client, to CICS as an HTTP server
N
N-> Requests to a server, from CICS as an HTTP client
N
N-> Requests from a SOAP client to CICS as a web service provider
N
N-> Requests to a web service provider from CICS as a web service
N   requester
N
N-> Requests from a web client to Java web application
N
N-> Atom feed requests
N
N-> Business event emissions from the HTTP EP adapter
N
N-> URIMAP search order
N
N-> URIMAP resources in CICS bundles
N
N-> URIMAP resources as application entry points
