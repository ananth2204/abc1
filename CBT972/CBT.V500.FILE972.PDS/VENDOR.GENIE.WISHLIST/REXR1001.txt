NChapter 10. TSO/E REXX commands
N-------------------------------
NDELSTACK
N--------
NDELSTACK deletes the most recently created data stack that was created
Nby the NEWSTACK command, and all elements on it. If a new data stack
Nwas not created, DELSTACK removes all the elements from the original
Ndata stack.
N
N
NSyntax:
N  ADDRESS TSO "DELSTACK"
N
N
NDELSTACK Example
N----------------
MSee REXR1009.
