NSyntax of Parameters
N====================
N
NSyntax rules
N------------
NSyntax rules define how to code the fields and parameters on job
Ncontrol statements. The syntax indicates:
N
N->What the system requires.
N
N->What is optional for the specific purpose or process you are
N  requesting.
N
N->How the parameters are to appear.
I
IThe syntax rules apply to all job control statements: JCL
Istatements, JES2 control statements, and JES3 control statements.
N
NBackward References
N-------------------
NMany parameters in job control statements can use a backward
Nreference to fill in information.
