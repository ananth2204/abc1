NChapter 19. MAPSET resources
N============================
NA MAPSET resource defines a BMS map sets.
N
NEach interactive application using a display device can use specific
Nscreen layouts, or maps. These are not specified in the program itself.
NInstead, you use basic mapping support (BMS). This gives greater
Nflexibility and allows the maps to be used by multiple invocations of
Nthe same program, or by several different programs. You specify maps,
Nand the fields on them, using the DFHMSD, DFHMDI, and DFHMDF macros.
N
NFor further guidance on this, see the
NSC34-7158-02 CICS Application Programming Guide.
