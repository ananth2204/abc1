      UNIXCMDB - TSO command to execute UNIX commands and to
                 display the results.

                 A version of an exec by John McKown.  Purpose
                 is to execute UNIX commands from TSO, without
                 using a UNIX shell.  I made this name longer
                 because there were several variants posted on
                 the MVS-OE list, but this one (or whichever
                 variant you really use) should be renamed to
                 be called UNIX.

      USSFILMT - TSO command to list UNIX files and how they
                 are mounted.

      USSGETMT - TSO command to display all mounted files
                 under UNIX.
