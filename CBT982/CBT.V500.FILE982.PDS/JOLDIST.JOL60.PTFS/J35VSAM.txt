* ON LATER VERSIONS OF THE MVS TYPE OPERATING SYSTEMS (MVS,F4)
*    IT IS ILLEGAL TO CODE VOLUME SERIAL INFORMATION FOR VSAM DATA
*    SETS.
*
* THE FOLLOWING ZAP FORCES JOL TO INVOKE SHOWCAT WHEN A CATALOG
*    HAS BEEN SEARCHED. IT THEN AUTOMATICALLY SETS THE VSAM INDICATOR
*    IF THE DATA SET IS VSAM.
*
 NAME UV00MN UJS01CAT
 BASE 0F38
 VER 1B72 BC3A,BC3C,BC3E,BC40     CHECK ZAP AREA CLEAR
 VER 1B82 BC4A,BC4C,BC4E,BC50     CHECK ZAP AREA CLEAR
 VER 10FA 4780,B272               CHECK CAT RETURN OF ZERO BRANCH
 REP 1B72 D713,2000,2000          CLEAR CALLAREA FOR CALL TO SHOWCAT
 REP 1B78 9280,2000               SET SHOWCAT PARM(1)
 REP 1B7C 4110,301C               LA  R1, DSNAME
 REP 1B80 5010,2004               SECOND PARAMETER SET UP
*                                 THIRD PARAMETER IS ZERO
 REP 1B84 4110,2EF8               LOAD ADDRESS WORK+800
 REP 1B88 5010,200C               THIRD PARM SET UP
 REP 1B8C 41F0,BCBE          SET UP LINK SVC NOW
 REP 1B90 50F0,BCBA               STORE PARAMETER ADDRESS
 REP 1B94 41F0,BCBA               LA ITS ADDRESS NOW
 REP 1B98 0A06                    SVC 6 (LINK TO SHOWCAT)
 REP 1B9A 12FF                    TEST RETURN CODE
 REP 1B9C 4770,B272               NON-ZERO MEANS NON-VSAM, PRESS ON
 REP 1BA0 9601,80A4               SET VSAM TYPE DATA SET
 REP 1BA4 47F0,B272               PRESS ON.
 REP 1BF6 0000,0000               SET ZEROS (NO TASKLIB)
 REP 1BFA D2C8C8F0C3D3C1F0        KHH0CLA0 (F4 SHOWCAT)
 IDRDATA VSAMCAT
