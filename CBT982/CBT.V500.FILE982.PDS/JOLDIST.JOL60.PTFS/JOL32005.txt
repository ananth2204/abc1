*JOL32005  APR 15,1980
*PROBLEM DESCRIPTION: WHEN READING A MACRO STATEMENT FROM THE MACRO
*   LIBRARY OF THE FORM:-
*      X:MACRO(%A %B %C %D);
*   IE IT DOES NOT HAVE COMMAS BETWWEN EACH PARAMETER, EVERY SECOND
*   PARAMETER IS MISSED.
*
*   THIS IS DUE TO JOL EXPECTING A COMMA BETWEEN EACH PARAMETER.
*
*SOLUTION:  STOP STORING THE TOKEN NUMBER AFTER FINDING SUCH A PARAMETER
*       AND CHECK LATER IF THERE IS A COMMA, AND IF SO, SKIP IT.
*FIX(IMASPZAP):
     NAME  UJP00MN  UJP53MAC
     BASE  0EE8
     VER  1126 40103002
     REP  1126 47000000
     IDRDATA JOL32005
*SUMMARY: MACRO-STMT PARAMETERS
*MACRO STATEMENT ERRORS PARAMETERS
