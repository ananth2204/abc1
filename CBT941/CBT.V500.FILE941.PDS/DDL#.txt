Function:


This REXX exec generates definition - IDCAMS DEFINE, ALLOCATE as well as
JCL DD  statements - for an existing data set, GDG base and VSAM
object.  The generated definition may be used to easily re-create
objects on the same or another z/OS system, that's may be especially
useful to reallocate data set on the target system when those are
transferred via ftp.

I hope, this procedure will supersede CBT tape File 920 by John McKown: it
is able to generate declaration of all kind of VSAM data sets as well as
any RFS object - non-VSAM dataset, AIX, path, GDG base, alias.


Syntax:


    DDL   hlq  ÝDCB(YES¦NO)¨  ÝSMS(YES¦NO¦ONLY)¨  ÝAMS(ALL¦VSAM)¨


    hlq     specifies the high-level qualifier(s) being processed.
            May be also a fully-qualified data set.

    DCB     YES   causes generation of all DCB attribites (RECFM,
                  LRECL, BLKSIZE);
            NO    suppresses generation of all DCB attribites.

    SMS     YES   causes generation of all SMS-related allocation
                  parameters (DATACLAS, MGMTCLAS etc) as well as
                  old-fashioned allocation parameters (VOLUME, UNIT)
            ONLY  causes generation of all SMS-related allocation
                  parameters (DATACLAS, MGMTCLAS etc) only
            NO    causes generation of old-fashion allocation
                  parameters (VOLUME, UNIT) only.

    AMS     ALL   generates AMS statements for object definition,
                  includung ALLOCATE;
            VSAM  generates AMS statements for VSAM and GDG and DD
                  statements for non-VSAM allocations.


Usage:


    This procedure can be invoked via JCL or interactively (ISPF 6
    or as a ISPF 3.4 prefix command).  An example of JCL is supplied,
    but needs to be changed according to data set naming conventions
    on your z/OS system.  When DDL is invoked interactively, it writes
    generated definitions to the temporary data set and then enters
    ISPF EDIT, so those may be easily saved.  In batch mode DDL
    writes generated output to DDL DD.
