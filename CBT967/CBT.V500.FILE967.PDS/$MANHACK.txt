While contributions to this collection must be in SMP/E usermod format,
an increasing number of people have requested instructions to manually
install each piece of CUCI.  So here is a list of components for each
fix, so simply copy each component into the DD listed for it (also
follow any additional directions listed).  Ensure that any members
matching IBM members appear in the concatenation before those members.

-----------------------------------------------------------------------
USRCCONF dialog for default ISPF settings not set in ISPCCONF:
-----------------------------------------------------------------------

ISPPLIB - ISR@PRIM (modify your ISR@PRIM member with CUCI changes)
          USRPCONF
          USRPMOD
          USRPMODQ
          USRPMOD1
          USRPMOD2
          USRPMOD3
          USRPMOD4
          USRUMNO2
          USRUULO2
          USR0Y002
          USR0Y004
          USR0Y007
SYSPROC - USR@KYWD (modify for your ISPF keyword file member)
          USRCCONF
          USRCCVWK
          USRCMOD

-----------------------------------------------------------------------

for highlighting (73 COBOL and 140 PL/I keywords added)
-----------------------------------------------------------------------

ISPLLIB - ISRPX (must be assembled and link-edited per instructions in
                 member ISRPXASM)
