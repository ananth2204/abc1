For loadlib PDS(/E)s, the "L" line command initiates a binder dialog.

The "sections" for the selected member will be displayed with a
line command for each section.

Line commands available are

 E    display ESD info
 B    display Binder IDR info
 M    display section "map"
 L    display language processor data
 R    display RLD info
 Z    display ZAP info
