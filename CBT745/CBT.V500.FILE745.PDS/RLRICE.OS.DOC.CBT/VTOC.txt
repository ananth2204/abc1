 When viewing a VTOC

 line commands are:
     B             Browse
     C             Catalog
     D             Delete
     I             Detail info
     R             Rename
     S             Select (same as browse)
     U             Uncatalog
     X             Compress


 Primary commands
     F             Find a data set
     L             Locate a data set (same as find)
     FIND          Find a data set
     LOCATE        Locate a data set (same as find)
