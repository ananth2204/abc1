)F FUNCTION - The CALLMODL exec is used to display Link-Edit (Binder)
              information for a load module in a specific load
              library.  The results are displayed on the screen.
)X SYNTAX   -

         CALLMODL   MODULENAME
         CM
                               { LIB(dsname)  }


      Modulename - The modulename to be searched.  This parameter
                   must always be specified and must precede any
                   subparameters which are also specified.


      Subparameter Values and Explanations:


      LIB(dsname)- This subparameter tells the facilitiy to search
                   this Load Library for the modulename.  If this
                   subparameter is omitted, the facility will search
                   in library 'SYS1.LINKLIB' as a default.
