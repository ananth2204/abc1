Special Notes
=============
The BASHPAXZ in this PDS is a _complete_ pax archive of the BASH 4.2 UNIX
directory. It contains not only the source and executable files, but all the
intermediary files, such as .o (object) files. This is by design. Yes,
it makes the archive larger. But it makes it absolutely complete.

Some of the files that you might be most interested in, at first, are
those which start with do_ . In particular, if you want to run the BASH
"IVP", run the job in do_tests.jcl . Of course, you will need to modify
this job depending on your shop standards, and with the name of the UNIX
directory into which you unwound the archive. (BTW - the word "unwound"
is archaic UNIX terminolog because archives were originally on physical
tape. Which was "wound" around reel. So you would "unwind" the archive.
At least, as it was told to me.)

If you just wanted the executables, you got much more than you really
needed.  In this case, all of the files you need are in the bash/dist
directory in this archive.
