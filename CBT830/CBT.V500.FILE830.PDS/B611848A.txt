CA-ACF2 and SAF


SAMPLE PROGRAM

RAC1     CSECT
RAC1     AMODE 31
RAC1     RMODE ANY
         USING RAC1,15
         BAKR  14,0
         B     START
         DC    C'RAC1 &SYSDATE &SYSTIME'
START    DS    0H
         DROP  15
         LR    12,15
         USING RAC1,12
         LA    13,SAVEAREA
         MVC   4(4,13),=C'F1SA'
         SPACE
         RACROUTE REQUEST=AUTH,                                        X
               CLASS=CLASS,                                            X
               ENTITY=ENTITY,                                          X
               WORKA=RACFWORK,                                         X
               MF=(E,RACAUTH)
         LTR   15,15
         BZ    RETURN
         DC H'0'
RETURN   DS    0H
         SLR   15,15
         PR
         LTORG
*---------------------------------------------------------------------*
* Data Areas                                                          *
*---------------------------------------------------------------------*
CLASS    DC    AL1(8)
         DC    CL8'APPLES'
ENTITY   DC    CL10'MACINTOSH'
RACAUTH  RACROUTE REQUEST=AUTH,MF=L    List form of RACROUTE macro
SAVEAREA DS    18F
RACFWORK DS    CL512
         END
